# Adirun retro style running game

This is simple browser based game created as a fun surprise for Aditya's 32nd birthday.

It is inspired by the original mario graphics.

Almost all the code has been created with the help of LLMs:

- [Gemini](https://gemini.google.com/) was used to create pixel style images and the sprite sheets. It was also used to get the initial skeletal code.
- The code was then iteratively improved using [Kimi K2.7 Code](https://www.moonshot.ai/) (via [OpenCode](https://opencode.ai/)).
